# FaleMais

FaleMais is an application that calculates the cost of two distinct area codes

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Flask and pytest.

```bash
pip install pytest
```
```bash
pip3 install flask==0.12.2
```

## Usage

```bash
python3 main.py
```
## Test

```bash
pytest
```
